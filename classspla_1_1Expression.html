Detailed Description

Computational expression for submission.

Represents single self-sufficient set of computational nodes with dependencies, which can be submitted for the execution at once and then asynchronously checked state of computation of waited (blocked) until completion.

Warning
Expression cannot be modified after submission.
Expression::Wait() is valid call only for submitted expressions.
Note
Call Expression::Wait() to ensure completion.
Call Expression::GetState() to get current state to ensure, that expression is not running.
If expression is released, it implicitly calls Expression::Wait() internally before destruction.
